Parser

Description

Takes a list of Token and do an syntaxic analysis, based on a BNF grammar, to build an Decl list representing the program Abstract Syntax Tree.

If an unexpected token is found or the syntax is invalid, the parser function returns an pretty :) error message.

parser :: [SingleToken] -> Either String [Decl] Source #

Takes a [SingleToken] as parameter and returns a Either String [Decl].

On success, this function returns a list of Decl representing the complete AST of the program.

On failure, this function returns a pretty formatted error message.
