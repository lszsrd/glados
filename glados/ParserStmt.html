parseCompoundStmt :: Decl -> Parser CompoundStmt Source #

Takes an Decl and a '[SingleToken]' as parameter and returns a Either String CompoundStmt.

On success, this function returns a Stmt.

On failure, this function returns a pretty formatted message error.

This function is used to parse a compound statement, (e.g.: a function's body, a for's body).
